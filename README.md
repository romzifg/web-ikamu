# IKA-UNM — Website Ikatan Alumni

Website resmi Ikatan Alumni Matematika Universitas Islam Bandung, dibangun dengan **Next.js 16**, **TypeScript**, dan **Tailwind CSS**.

## 🚀 Cara Menjalankan

### 1. Install dependencies
```bash
npm install
```

### 2. Jalankan development server
```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

### 3. Build untuk production
```bash
npm run build
npm start
```
